Il est possible de lancer un code avec un temps de retard :

setTimeout() - clearTimeout()

la fonction setTimeout() permet de lancer un code avec un retard indiqué en millisecondes

la fonction setTimeout retourne un identifiant unique au moment de son execution. Cet identifiant permet d'annuler l'action en utilisant clearTimeout

setInterval() - clearInterval()

la fonction setInterval permet de lancer un code à interval régulier. Le temps est indiqué en millisecondes
